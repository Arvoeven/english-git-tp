Git cheat sheet

Init

Create a repository in a folder

git init

Clone

Clone a repository in a new folder

git clone git@github.com:Luuka/english-git-tp.git

Create a branch

Create a branch and jump to it

git checkout -b "your-branch"

Jump to another branch

List all existing branchs

git branch

Checkout to another branch

git checkout your-branch

Prepare a commit

Check modifications

git diff

Add changed file(s) to the commit

git add ./path/to/file

Or all modified files

git add -A

Commit with a message

git commit -m "Your commit message"

Modify a non-published commit

Add new modifications to last commit

git commit --amend

Git will open your previous commit message in a text editor. Edit your commit message and exit the editor.

Your commit message

# Veuillez saisir le message de validation pour vos modifications. Les lignes
# commençant par '#' seront ignorées, et un message vide abandonne la validatio$
#
# Date :       Wed Oct 25 09:50:12 2017 +0200
#
# Sur la branche master
# Votre branche est en avance sur 'origin/master' de 1 commit.
#   (utilisez "git push" pour publier vos commits locaux)
#

#

Publish your modifications

Push the origin branch to the remote branch

git push origin your-branch

Merge a branch

Checkout the branch where you want to merge another branch

git checkout master

Merge a branch into the current branch

git merge your-branch

Update your branch

Prepare your modifications

git add -A

Put your changes aside

git stash save

Download remote modifications on origin branchs

git fetch origin

Apply changes from remote to local branch

git rebase origin/your-branch

Reapply your modifications

git stash apply
