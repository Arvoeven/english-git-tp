Git cheat sheet

Init

Create a repository in a folder

git init

Clone

Clone a repository in a new folder

git clone git@github.com:Luuka/english-git-tp.git

Prepare a commit

Add changed file(s) to the commit

git add ./path/to/file

Or all modified files

git add -A

Commit with a message

git commit -m "Your commit message"

Modify a non-published commit

Add new modifications to last commit

git commit --amend
